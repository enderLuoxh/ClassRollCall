## 📌 项目简介

ClassRollCall 是一款轻量级、美观实用的班级点名工具。它悬浮在屏幕上的半透明按钮，支持拖动、自定义样式，抽取名字时会显示动画和毛玻璃效果弹窗。适合教师课堂点名、活动抽选等场景。

## ✨ 功能特性

### 🎯 核心功能
- **一键抽取**：点击按钮随机抽取名单中的名字
- **两种抽取模式**：
  - 默认模式：抽完一轮后重新洗牌（保证每人被抽中次数均衡）
  - 真随机模式：每次抽取后重新洗牌（完全随机）

### 🎨 视觉特效
- **全屏动画效果**
- **毛玻璃弹窗**：
  - 大字号居中显示被点名字
  - 3秒后自动消失
  - 支持自定义字体、颜色、透明度
  - 可设置背景图片

### 🎛️ 高度自定义
- **按钮样式**：
  - 纯色、渐变色、金属质感
  - 可自定义颜色和透明度
  - 可修改按钮文字
- **弹窗样式**：
  - 纯色、渐变色、金属质感背景
  - 自定义文字颜色、字体
  - 支持背景图片（可调节透明度）
- **开机自启动**：可选开机自动运行

### 🖱️ 便捷操作
- **拖动放置**：可随意拖动按钮到屏幕任意位置
- **位置记忆**：自动保存上次关闭时的位置
- **位置重置**：若拖出屏幕范围，可一键重置到右下角
- **系统托盘**：最小化到托盘，后台运行不干扰

## 📋 使用方法

### 准备工作
1. 在程序目录下创建 `names.csv` 文件（或通过设置选择自定义文件）
2. 按以下格式添加名字（每行一个）：
```csv
张三
李四
王五
赵六
```

### 基本操作
- **左键点击按钮**：抽取名字
- **按住按钮拖动**：移动按钮位置
- **右键点击按钮**：弹出菜单（显示/隐藏窗口、设置、关于、退出）
- **双击托盘图标**：显示/隐藏窗口

### 设置说明
1. 右键点击按钮 → 选择"设置"
2. 在设置对话框中可配置：
   - **常规**：开机自启动、抽取模式、动画效果、名单文件、位置重置
   - **按钮**：文字、透明度、颜色/渐变/金属样式
   - **弹窗**：字体、文字颜色、背景样式、背景图片

## 🚀 安装运行

### 方法一：直接运行（需要Python环境）
```bash
# 安装依赖
pip install PyQt5

# 运行程序
python ClassRollCall.py
```

### 方法二：打包成exe
```bash
# 安装pyinstaller
pip install pyinstaller

# 打包
pyinstaller --onefile --windowed --name "ClassRollCall" ClassRollCall.py
```

程序会自动在 `%LOCALAPPDATA%\ClassRollCall\` 目录下创建配置文件：
- `settings.json`：保存所有设置
- `position.json`：保存按钮位置
