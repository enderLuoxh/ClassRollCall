## 📌 项目简介

ClassRollCall 是一款轻量级、美观实用的班级点名工具。它悬浮在屏幕上的半透明按钮，支持拖动、自定义样式，抽取名字时会显示炫酷的全屏动画和毛玻璃效果弹窗。适合教师课堂点名、活动抽选等场景。

## ✨ 功能特性

### 🎯 核心功能
- **一键抽取**：点击按钮随机抽取名单中的名字
- **两种抽取模式**：
  - 默认模式：抽完一轮后重新洗牌（保证每人被抽中次数均衡）
  - 真随机模式：每次抽取后重新洗牌（完全随机）

### 🎨 视觉特效
- **全屏动画效果**（可切换）：
  - 简约波纹：优雅的水波扩散
  - 星光效果：8个方向的星光射线
  - 旋转光环：旋转的椭圆光环
  - 无动画：简洁模式
- **毛玻璃弹窗**：
  - 大字号居中显示被点名字
  - 3秒后自动消失
  - 支持自定义字体、颜色、透明度
  - 可设置背景图片

### 🎛️ 高度自定义
- **按钮样式**：
  - 纯色、渐变色、金属质感
  - 可自定义颜色和透明度
  - 可修改按钮文字
- **弹窗样式**：
  - 纯色、渐变色、金属质感背景
  - 自定义文字颜色、字体
  - 支持背景图片（可调节透明度）
- **开机自启动**：可选开机自动运行

### 🖱️ 便捷操作
- **拖动放置**：可随意拖动按钮到屏幕任意位置
- **位置记忆**：自动保存上次关闭时的位置
- **位置重置**：若拖出屏幕范围，可一键重置到右下角
- **系统托盘**：最小化到托盘，后台运行不干扰

## 📋 使用方法

### 准备工作
1. 在程序目录下创建 `names.csv` 文件（或通过设置选择自定义文件）
2. 按以下格式添加名字（每行一个）：
```csv
张三
李四
王五
赵六
```

### 基本操作
- **左键点击按钮**：抽取名字
- **按住按钮拖动**：移动按钮位置
- **右键点击按钮**：弹出菜单（显示/隐藏窗口、设置、关于、退出）
- **双击托盘图标**：显示/隐藏窗口

### 设置说明
1. 右键点击按钮 → 选择"设置"
2. 在设置对话框中可配置：
   - **常规**：开机自启动、抽取模式、动画效果、名单文件、位置重置
   - **按钮**：文字、透明度、颜色/渐变/金属样式
   - **弹窗**：字体、文字颜色、背景样式、背景图片

## 🚀 安装运行

### 方法一：直接运行（需要Python环境）
```bash
# 安装依赖
pip install PyQt5

# 运行程序
python name_card.py
```

### 方法二：打包成exe
```bash
# 安装pyinstaller
pip install pyinstaller

# 打包
pyinstaller --onefile --windowed --name "点名器" name_card.py
```

## 📁 文件结构
```
ClassRollCall/
├── name_card.py          # 主程序
├── names.csv             # 默认名单文件（自动生成）
└── README.md             # 说明文档
```

程序会自动在 `%LOCALAPPDATA%\ClassRollCall\` 目录下创建配置文件：
- `settings.json`：保存所有设置
- `position.json`：保存按钮位置

## ⚙️ 配置文件说明

### settings.json 主要配置项
```json
{
    "auto_start": false,           // 开机自启动
    "pick_mode": "default",         // 抽取模式
    "animation_style": "ripple",    // 动画样式
    "button_text": "Call",          // 按钮文字
    "button_style": "solid",        // 按钮样式
    "button_color1": [100,150,255], // 按钮颜色1
    "button_color2": [200,200,255], // 按钮颜色2
    "popup_font_family": "Microsoft YaHei", // 弹窗字体
    "popup_text_color": [40,60,120],        // 文字颜色
    "popup_bg_style": "solid",               // 弹窗背景样式
    "popup_bg_opacity": 220,                  // 背景透明度
    "popup_use_bg_image": false               // 是否使用背景图片
}
```
